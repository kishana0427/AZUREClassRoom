## Storage Options

### Harddisk
* Two Options
    * Storage Account
        * one account for all kinds of storage
    * Managed Disk
        * specially designed only for vm harddisks
* Two Hardware Types
    * HDD
    * SSD
* Speed of Hardisk
    * IOPS
        * 1 IOPS => 256 kb/s
* Cost Perspective
* VM => Multiple Harddisks
    * Harddisk => OS => Primary Disk/Root disk
    * Secondary Disk
    * Temporary DataDisk
* Backup of Harddisk
    * Snapshot 
* PLANS
    * Basic Plans
    * Replication Plans
